#tail(other) ⇒ Array Also known as: /

Returns an Array with the Object being the head and the other Object being the tail.

Examples:

'head' / ['body', 'foot']   # ==> ['head', 'body', 'foot']
7.tail(:t)                  # ==> [7, *:t]

Parameters:

  • other (Object) —

    the Object which is to be the tail
